Introdução ao JQuery

Seletores

Ferramentas para selecionar e manipular elementos HTML de forma eficiente.

Básicos

 $("p") => Seleciona todos os paragráfos.
$("#id") => Seleciona elemento com id específico.
$(".classe") => Seleciona todos os elementos com aquela classe.

Avançados

 $("div > p") => Seleciona o filho direto da div, no caso o primeiro paragráfo.
$("[attribute]") => Seleciona todos os elementos com determinado atributo.
$("ul li:first") => Seleciona o primeiro item da lista.
